01 - (WORKING DIRECTORY)
  Trabajo los archivos en local (WORKING DIRECTORY) hasta que decida que quiero pasarlos a espera (STATING AREA) para hacer un Commit (REPOSITORY).
02 - (STATING AREA)
  git status (verde: salvados en SA/rojo: no salvados en SA)
  git add nombreDelArchivo.extensiong para ir salvando (en STATING AREA) los archivos como los quiera guardar en un commit (REPOSITORY).
03 - (REPOSITORY)
  para hacer el Commit y congelar una version los archivos (REPOSITORY) que en 01 (WORKING DIRECTORY) salve con git add nombreDelArchivo en 02 (STATING AREA)git 
COMANDOS
 pwd - para ver la ruta en la que estamos
 git init - instalar Git en esa carpeta
 git status para ver el estado de los archivos (seguidos/no seguidos)
 git add nombreDelArchivo.extension para agregar archivo al stating area
 git commit -m "comentarioDelCommit
 git checkout -- nomnreDeArchivo.extension para volver el archivo para atras
 git diff para ver las diferencias hechas en los archivos
 git log para listar los commit
 git push -u origin master para subirlo a github.
